过滤选择器

  • 基本过滤选择器

    • :first 选择第一个元素 $(“div:first”) -> 选择所有 div 元素中的第一个 div 元素
    • :last 选择最后一个元素 $(“div:last”) -> 选择所有 div 元素中的最后一个 div 元素
    • :not(selector) 去除所有与给定选择器匹配的元素 $(“input:not(.myClass)”) 选取 class 不是 myClass 的 input 元素
    • :even 选择索引值是偶数的所有元素, 索引从 0 开始 $(“input:even”) -> 选择索引是偶数的 input 元素
    • :eq(index) 选取索引等于 index 的元素 $(“input:eq(1)”) 选取索引值为1的 input 元素
    • :gt(index) 选取索引大于 index 的元素 $(“input”:gt(1)) 选取索引值大于1的 input 元素
    • :lt(index) 选取索引小于 index 的元素 $(“input:lt(1)”) 选取索引值小于1的 input 元素
    • :header 选取所有的标题元素 $(“:header”) 选取所有的标题元素 h1, h2, h3 …
    • :animated 选取正在执行动画的所有元素 $(“div:animated”) 选取所有的正在执行动画的 div 元素

  • 内容过滤选择器

    • :containers(text) 选取文本内容包含 “text” 的元素 $(“div:containers(‘我;)”) 选择含有文本 “我” 的 div 元素
    • :empty 选取不包含子元素或者文本的空元素 $(“div:empty”) 选取不包含子元素的 div 元素
    • :has(selector) 选取含有选择器所匹配的元素 $(“div:has(p)”) 选取含有 p 元素的 div 元素
    • :parent 选择含有子元素或者文本的元素 $(“div:parent”) 选择拥有子元素的 div 元素

  • 可见性过滤选择器

    • :hidden 选取所有的不可见元素 $(“:hidden”) 选取所有的不可见元素
    • :visible 选取所有的可见元素

  • 属性过滤选择器

    • [attribute] 选择拥有此属性的元素 $(“div[id]”) => 选择拥有 id 属性的 div 元素
    • [attribute=value] 选择属性值为 value 的元素 $(“div[title=test]”) => 选择属性 title 为 test 的 div 元素
    • [attribute!=value] 选择属性值不为 value 的元素 $(“div[title!=test]”) => 选择属性 title 不为 test 的 div 元素
    • [attribute^=value] 选择属性值以 value 开始的元素 $(“div[title^=test]”) => 选择属性 title 以 test 开始的 div 元素
    • [attribute$=value] 选择属性值以 value 结束的元素 $(“div[title$=test]”) => 选择属性 title 以 test 结束的 div 元素
    • [attribute*=value] 选择属性值包含 value 的元素 $(“div[title*=test]”) => 选择属性 title 包含 test 的 div 元素

  • 子元素选择器

    • :nth-child(index/even/odd/quation) 选择每个父元素下的子元素
      • nth-child(2) 选取每个父元素下索引值为2的子元素
      • nth-child(odd) 选取每个父元素下索引值为奇数的子元素
      • nth-child(3n) 选取每个父元素下索引值为 3 的倍数的子元素 (n从0开始)
    • :first-child 选择每个父元素下的第一个子元素
    • :last-child 选择每个父元素下的最后一个子元素
    • :only-child 如果某个元素是它父元素的唯一子元素, 那么它将被匹配

  • 表单对象属性过滤选择器

    • :enabled 选择所有可用元素 $(“#form :enabled”) => 选择 id 为 form 的表单中所有的可用元素
    • :disabled 选择所有的不可用元素 $(“#form :disabled”) => 选择 id 为 form 的表单中的所有的不可用元素
    • :checked 选择所有的被选中的元素 $(“input:checked”) => 选择所有被选中的 input 元素
    • :selected 选择所有被选中的选项元素(下拉列表) $(“select :selected”) => 选取所有的被选中的元素

  • 表单选择器

    • :input 选择所有的 input, textarea, select, button 元素
    • :text 选择所有的单行文本框
    • :password 选择所有的密码框
    • :radio 选择所有的单选框
    • :checkbox 选择所有的复选框
    • :submit 选择所有的提交按钮
    • :images 选择所有的图像按钮
    • :reset 选择所有的重置按钮
    • :button 选择所有的按钮
    • :file 选择所有的上传域
    • :hidden 选择所有的不可见元素
